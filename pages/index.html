FTL: Faster Than Light

FTL: Faster Than Light is a real-time strategy roguelike game created by indie developer Subset Games,
which was released for Microsoft Windows, macOS and Linux in September 2012.


With advanced edition enabled there are 28 different ships.
The first ship you start with is The Kestrel Cruiser.

Other ships:

  • The Engi Cruiser
  • The Federation Cruiser
  • The Zoltan Cruiser
